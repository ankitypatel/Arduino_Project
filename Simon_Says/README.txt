This is Simon Says game using Teensy 3.1 with 16*2 LCD display.

push button 1 -> PIN 1
push button 2 -> PIN 12
push button 3 -> PIN 13
push button 4 -> PIN 23

LED 1 -> PIN 2
LED 2 -> PIN 11
LED 3 -> PIN 14
LED 4 -> PIN 21

LCD connection with Teensy 3.1, Please refer:
https://www.pjrc.com/teensy/td_libs_LiquidCrystal.html
